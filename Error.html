Semantic HTML Corrections

1. Missing Alt Text

Error: <img src="pic.jpg">

Fix: <img src="pic.jpg" alt="Description">

2. Block inside Inline

Error: <a><div>...</div></a> (Pre-HTML5 issue)

Fix: Use CSS for block display.

3. Non-Unique IDs

Error: id="nav" used twice.

Fix: Use class="nav" instead.

4. Skipped Heading Levels

Error: h1 followed by h4.

Fix: Use h1, then h2, then h3.

5. Missing Form Labels

Error: Input without <label>.

Fix: Always link label via for="id".

6. Deprecated Tags

Error: <center>, <font>.

Fix: Use CSS for alignment and styling.

7. Empty Links

Error: <a href="#"></a>.

Fix: Add text or aria-label.

8. Justified Text

Error: align="justify".

Fix: Avoid (bad for dyslexic users).

9. Div Soup

Error: Using div for everything.

Fix: Use main, article, section.

10. Missing Lang Attribute

Error: <html>.

Fix: <html lang="en">.
